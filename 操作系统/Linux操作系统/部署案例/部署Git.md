# 部署Git

## 1.官网下载地址

https://git-scm.com/

## 2.命令下载

```sh
yum install git
```

## 3.检查安装

```sh
git version
```

## 4.配置个人信息

```sh
[root@localhost wqy]# git config --global user.name "Devotion"
[root@localhost wqy]# git config --global user.email "2168149199@qq.com"
[root@localhost wqy]# git config -l
user.name=Devotion
user.email=2168149199@qq.com
```

### 其余命令与gitbash相似！！！
