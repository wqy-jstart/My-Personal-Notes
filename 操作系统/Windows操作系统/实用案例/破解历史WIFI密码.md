# 破解历史WIFI密码

### 1.`win + R` 打开运行框进入cmd

```sh
win + R
cmd
```

### 2.输入命令

```sh
netsh wlan show profiles
```

显示历史连接的WIFI:

```sh
接口 WLAN 上的配置文件:


组策略配置文件(只读)
---------------------------------
    <无>

用户配置文件
-------------
    所有用户配置文件 : 2的1000次方取前八位。
    所有用户配置文件 : TP-LINK_304
```

### 3.输入命令：

```sh
netsh wlan show profiles "TP-LINK_304" key=clear
```

显示该WIFI信息包括密码：

=======================================================================
